# MVPA60

## Dependencies
- Node.js
- Yarn
- Expo
- React Native
- Supabase

## Before you start

- Ensure you have installed Node.js, Yarn, ni(if need).
- If you want to run on ios simulator, you need to install xcode and simulator.

## Run Client

```bash
cd ./expo
yarn install # or ni
npx expo # or nr dev
```
